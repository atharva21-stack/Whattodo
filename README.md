# Note Taking App 📒✏

This is simple note taking build with React Js 

## Run Locally

Clone the project

```bash
  git clone https://github.com/atharva21-stack/Whattodo
```

Go to the project directory

```bash
  cd Whattodo
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** Node, Express
